# Projeto Carrinho de Compras


# 📌 Sobre o projeto

Esse projeto tem como objetivo criar um carrinho de compras utilizando HTML, CSS e JavaScript, com ênfase no treino de manipulação do DOM e no desenvolvimento da lógica de programação.

A ideia foi desenvolver uma aplicação simples, na qual o usuário possa adicionar produtos ao carrinho, visualizar os itens escolhidos e calcular o valor total da compra.

Este projeto foi desenvolvido como parte do currículo do curso **Técnico em Desenvolvimento de Sistemas** da Escola **SENAI A. Jacob Lafer**.
---

## ⚙️ Funcionalidades e Conceitos Aplicados

Durante o desenvolvimento, vão ser aplicados os seguintes conceitos técnicos:

* Adicionar produtos ao carrinho

* Exibir lista de produtos selecionados

* Atualizar quantidade de itens

 * Remover itens do carrinho

* Calcular o valor total automaticamente
